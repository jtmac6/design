1. Run MIV.jar
2. Click File menu
3. Click Open submenu
4. Select folder with medical images (study)
5. Click choose

Press Prev button to scroll to previous image(s).
Press Next button to scroll to next image(s).
Press One button to change display state to one image.
Press Four button to change display state to four tiled images. 
Press File -> Exit menu to terminate program.
